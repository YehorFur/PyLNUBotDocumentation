Handles the /node_id_by_name command to find a node ID by its name prefix.

This function receives a node name prefix as an argument, searches for a node that matches the prefix in the specified building's graph, and returns the node's ID. If the node is found, the ID is sent to the user. If not, the user is notified that the node could not be found.

Handles the sending of photos for a selected building.

This function processes the callback query when a user selects a building. It finds the appropriate photos in the building's directory, converts any SVG files to PNG, and sends the media as a group. In case of errors during the process, the user is notified.
