Detailed Description

Contains Telegram bot command handlers for interacting with users.

This module defines several async functions for handling user commands, including sending photos, selecting buildings, and generating paths between nodes within buildings. The functions make use of the Telegram Bot API and integrate with other modules for processing images, generating paths, and handling building-related data.
